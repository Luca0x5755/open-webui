# PDF 引用來源面板

在知識庫（RAG）回答中，點擊指向 PDF 的精確引用，會在對話右側開啟該 PDF，並自動跳到被引用的頁面。

---

## 使用方式

1. 上傳 PDF 作為對話的知識來源，正常提問。
2. 回答中的引用標記若為精確格式（例：`[1#0]`），點擊它。
3. 右側開啟來源面板，顯示該 PDF；若來源含頁碼資訊，直接捲動到該頁。

## 行為對照

| 點擊的引用 | 結果 |
| --- | --- |
| 精確 PDF 引用（如 `[1#0]`），且來源有頁碼 | 右側面板開啟 PDF 並跳到該頁 |
| 精確 PDF 引用，但來源無頁碼 | 右側面板開啟 PDF（不跳頁） |
| 已開啟的同一份 PDF、點擊另一筆精確引用 | 直接更新到新頁，不重新載入 |
| 捲離後再次點擊同一引用 | 重新捲回被引用的頁面 |
| 一般引用 `[1]` | 沿用原本的引用彈窗 |
| 非 PDF 來源、或無效後綴（如 `[1#bad]`、`[1#999]`、`[1#]`） | 沿用原本的引用彈窗 |
| 回答最下方的「來源」清單 | 沿用原本的引用彈窗 |

## 適用範圍

- 適用於知識庫／來源引用（含以 PDF 檔案作為對話知識來源）。
- 右側面板可調整寬度；PDF 的縮放、重設等既有功能不受影響。

## 限制

- 不提供引用內容的高亮或框選。
- 僅支援 PDF，不支援其他檔案類型的側邊檢視。
- 引用落在同一頁時，畫面可能看不出差異。
